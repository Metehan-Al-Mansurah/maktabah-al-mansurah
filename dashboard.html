Einrichtung

Damit du und Oualid die gleiche Liste seht, brauchen wir einen kostenlosen Cloud-Speicher. Das ist einmalig in 3 Minuten erledigt:

  1. Geh auf jsonbin.io und erstell einen kostenlosen Account
  2. Klick oben rechts auf dein Profil → API Keys
  3. Kopier deinen Master Key (langer Code, beginnt mit $2a$...)
  4. Füg ihn hier ein und klick auf "Verbinden"

Dein Schlüssel wird nur lokal in diesem Browser gespeichert. Oualid braucht den gleichen Schlüssel — schick ihn ihm einfach.
